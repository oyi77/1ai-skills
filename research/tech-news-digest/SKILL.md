# Multi-Source Tech News Digest

**Aggregate tech news from 100+ sources, score by engagement, deduplicate, summarize via LLM, and deliver daily via Telegram.**

---

## Description

Automated tech news aggregation pipeline that pulls from RSS feeds, Twitter/X, GitHub Trending, Hacker News, and web search. Deduplicates stories, scores by engagement metrics, generates LLM-powered summaries, and delivers a curated daily digest to Telegram.

## Sources (100+)

### RSS Feeds
- TechCrunch, The Verge, Ars Technica, Wired
- Hacker News (top/best), Lobste.rs
- AI/ML: MIT Tech Review, arXiv CS daily, Hugging Face blog
- Dev: Dev.to, Changelog, InfoQ
- Crypto: CoinDesk, The Block, Decrypt

### Social / Real-Time
- Twitter/X: curated lists, trending hashtags, key accounts
- GitHub: trending repos (daily/weekly), release notifications
- Reddit: r/technology, r/programming, r/MachineLearning top posts

### Web Search
- DuckDuckGo news search for breaking stories
- Google News RSS for specific topics

## Pipeline

```
[Fetch Sources] → [Score by Engagement] → [Deduplicate] → [LLM Summarize] → [Deliver via Telegram]
```

### Step 1: Fetch
- RSS via `feedparser`
- Twitter via `twitter-cli` (uses TWITTER_AUTH_TOKEN + TWITTER_CT0 from ~/.bashrc)
- GitHub trending via web scraping / API
- HN via official API (`https://hacker-news.firebaseio.com/v0/`)
- Web search via `duckduckgo-search`

### Step 2: Score
Each story scored 0-100 based on:
- Source authority (tier 1/2/3)
- Engagement metrics (upvotes, retweets, stars)
- Recency (decay factor for older stories)
- Topic relevance (keyword matching against interests)

### Step 3: Deduplicate
- URL normalization
- Title similarity (fuzzy match, threshold 0.85)
- Merge engagement scores for same story across sources

### Step 4: LLM Summarize
- Top 15-20 stories sent to LLM for digest generation
- OmniRoute LLM: `base_url="http://localhost:20128/v1"`, `api_key="omniroute"`
- Output: 2-3 sentence summary per story + overall trend analysis

### Step 5: Deliver
- Format as Telegram message (markdown)
- Send via `openclaw message` tool or Telegram Bot API
- Archive to `memory/YYYY-MM-DD.md`

## Usage

```bash
# Run daily digest
python scripts/tech_news_digest.py

# Run with specific topics
python scripts/tech_news_digest.py --topics "AI,crypto,trading,opensource"

# Dry run (no Telegram delivery)
python scripts/tech_news_digest.py --dry-run

# Custom source count
python scripts/tech_news_digest.py --top 20
```
