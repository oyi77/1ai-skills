# Custom Morning Brief

**Daily briefing agent — aggregates news, tasks, open loops, and content drafts, delivered via Telegram.**

---

## Description

Automated morning briefing that pulls from multiple internal and external sources to create a personalized daily digest. Designed to give you a complete picture of what needs attention before you start your day.

## Sources

### Internal
- `memory/YYYY-MM-DD.md` — yesterday's notes and decisions
- `notes/open-loops.md` — unfinished tasks and pending items
- Recent git activity — commits, branches in progress

### External
- `duckduckgo-search` — breaking news in your focus areas
- `twitter-cli` — trending topics and mentions (uses TWITTER_AUTH_TOKEN + TWITTER_CT0 from ~/.bashrc)
- Market data — relevant prices and movements

---

## Pipeline

```
[Gather Sources] → [LLM Synthesize] → [Format Brief] → [Send to Telegram]
```

### Step 1: Gather
- Read yesterday's memory file
- Read open-loops.md for pending tasks
- Search news for configured topics
- Pull Twitter trends for relevant keywords

### Step 2: Synthesize
- OmniRoute LLM: `base_url="http://localhost:20128/v1"`, `api_key="omniroute"`
- Prioritize by urgency and relevance
- Flag items that need immediate attention

### Step 3: Format
- Structured Telegram message (markdown)
- Sections: Tasks, News, Market, Content Ideas

### Step 4: Deliver
- Send via `openclaw system event` (same pattern as HEARTBEAT.md)
- Archive brief to `memory/YYYY-MM-DD.md`

---

## Usage

```bash
# Run morning brief
python scripts/morning_brief.py

# Dry run (print to stdout, no Telegram)
python scripts/morning_brief.py --dry-run

# Custom topics
python scripts/morning_brief.py --topics "AI,trading,crypto,SEO"

# Specify memory date
python scripts/morning_brief.py --date 2026-03-20
```
