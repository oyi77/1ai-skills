## Hallucination in Data Extraction
- **Description**: Inventing fields that don't exist in the source text.
- **Root Cause**: Model trying to be "helpful" by filling gaps.
- **Fix**: Enforce "Extract ONLY what is explicitly stated" in system prompt.
