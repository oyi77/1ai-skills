## Comment (first comment)

I built this because my AI coding agent kept saying "I'll add tests later" and never doing it.

1ai-skills is a library of 1337 SKILL.md files that force AI agents to follow real workflows. Every skill includes:

1. **Anti-Rationalization Table** — A structured argument that prevents the agent from using common excuses (e.g., "Tests slow me down" → "Bugs slow you down 10x more")

2. **Code Examples** — Concrete, runnable code in Python, JS, Bash, SQL

3. **Workflow** — Step-by-step instructions with bold labels

4. **Verification Checklist** — Success criteria you can check

5. **When to Use / When NOT to Use** — Trigger conditions and contraindications

**19 categories:** Cybersecurity (786), Development (92), Content (64), Mindset (55), Marketing (45), Core (44), Integrations (36), DevOps (33), and 11 more.

**Quality:** 8-dimension test suite, 1337/1337 pass, 0 warnings, 0 failures. Every skill is structurally validated, has anti-rationalization tables, code examples, and verification checklists.

**Compatible with:** Claude Code, Cursor, Gemini CLI, Windsurf, OpenCode, GitHub Copilot, Kiro, and any agent that reads SKILL.md files.

This is complementary to addyosmani/agent-skills (24 skills, 67K stars) — his skills are engineering lifecycle commands (/spec, /build, /test), mine are domain-specific knowledge (cybersecurity, trading, marketing, etc.). Use both.
