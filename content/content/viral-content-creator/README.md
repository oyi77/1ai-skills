## ⚡ Quick Start (5 Minutes)

```bash
# 1. Generate 50 videos from 1 product image
python scripts/generate_videos.py \
  --image produk.jpg \
  --count 50 \
  --all-platforms

# 2. Filter by viral score (>60 only)
python scripts/viral_predictor.py --filter-score 60

# 3. Upload to PostBridge with scheduling
python scripts/postbridge_uploader.py \
  --campaign product_launch \
  --auto-activate

# 4. Start autopilot for 7 days
python scripts/autopilot.py --days 7
```

**Done!** You now have 50+ videos auto-posting across 5 platforms.
