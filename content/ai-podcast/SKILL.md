# AI Podcast Skill

## Overview

Create fully automated AI podcasts that research, write, and narrate complete episodes. Build a podcast network with multiple shows in different niches - all automated once set up.

**Time to Create**: 15 min per episode  
**Revenue**: $500-10K/month  
** Monetization**: Sponsorships, affiliates, premium content

---

## When to Use

- Build podcast presence without recording
- Create content at scale
- Test multiple niches quickly
- Repurpose existing content
- News/digest style shows

---

## When NOT to Use

- Personal brand podcasts
- Interview shows
- Live discussions
- High-production value shows

## Workflow

Step-by-step ai-podcast execution process.

**Step 1: Configure** — Set up targets and parameters in config file.

**Step 2: Execute** — Run the ai-podcast workflow with configured inputs.

**Step 3: Review** — Analyze outputs and iterate on configuration.

**Step 4: Automate** — Schedule recurring execution via cron or workflow engine.


### Step 1: Configure
Set up targets and parameters in config file.

### Step 2: Execute
Run the ai-podcast workflow with configured inputs.

### Step 3: Review
Analyze outputs and iterate on configuration.

### Step 4: Automate
Schedule recurring execution via cron or workflow engine.


### Stage 1: Research (5 min)
```
1. Pick topic or trend
2. AI researches the topic
3. Gather key points
4. Find supporting data
```

### Stage 2: Script (5 min)
```
1. Generate outline
2. Write introduction
3. Create main content
4. Write conclusion
5. Add transitions
```

### Stage 3: Generate Audio (5 min)
```
1. Select voices
2. Generate narration
3. Add sound effects
4. Mix audio
5. Add music
```

### Stage 4: Publish (2 min)
```
1. Upload to host
2. Generate show notes
3. Create timestamps
4. Submit to directories
```
