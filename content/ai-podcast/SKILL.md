# AI Podcast Skill

## Overview

Create fully automated AI podcasts that research, write, and narrate complete episodes. Build a podcast network with multiple shows in different niches - all automated once set up.

**Time to Create**: 15 min per episode  
**Revenue**: $500-10K/month  
** Monetization**: Sponsorships, affiliates, premium content

---

## When to Use

- Build podcast presence without recording
- Create content at scale
- Test multiple niches quickly
- Repurpose existing content
- News/digest style shows

---

## When NOT to Use

- Personal brand podcasts
- Interview shows
- Live discussions
- High-production value shows

## Workflow

### Stage 1: Research (5 min)
```
1. Pick topic or trend
2. AI researches the topic
3. Gather key points
4. Find supporting data
```

### Stage 2: Script (5 min)
```
1. Generate outline
2. Write introduction
3. Create main content
4. Write conclusion
5. Add transitions
```

### Stage 3: Generate Audio (5 min)
```
1. Select voices
2. Generate narration
3. Add sound effects
4. Mix audio
5. Add music
```

### Stage 4: Publish (2 min)
```
1. Upload to host
2. Generate show notes
3. Create timestamps
4. Submit to directories
```

## Monetization

### Sponsorships ($500-5K/episode)
```
Requirements: 1K+ downloads
Rates: $50-500 per 1K listeners
Niche: Tech, business, AI = premium
```

### Affiliate Links
```
Products: Equipment, software, services
Placement: Mention in episode
Commission: 10-30%
```

### Premium Content
```
Model: Paid feed + free preview
Price: $5-15/mo
Perk: Ad-free, bonus episodes
```

### Donations
```
Platform: Patreon, Buy Me Coffee
Average: $200-2K/mo for established shows
```
