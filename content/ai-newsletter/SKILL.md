# AI Newsletter Skill

## Overview

Build and monetize email newsletters using AI for content creation, curation, and automation. Create specialized newsletters in any niche - from AI trends to industry news to personal development.

**Revenue**: $1K-20K/month  
**Growth**: 20-50%/month possible  
**Time**: 30 min-2 hours/week after setup

---

## When to Use

- Build direct audience relationship
- Establish thought leadership
- Monetize expertise
- Create recurring revenue

---

## When NOT to Use

- No time for audience building
- Can't commit to consistency
- Very narrow niche

## Newsletter Types

### 1. Curated Newsletter
```
Format: Links + commentary
Frequency: Daily/Weekly
Example: Morning Brew, TLDR
Revenue: Ads, sponsorships
```

### 2. Original Content
```
Format: Deep essays
Frequency: Weekly
Example: Lenny's Newsletter
Revenue: Subscriptions
```

### 3. Hybrid
```
Format: Curation + original
Frequency: 2-3x/week
Example: Most creator newsletters
Revenue: Mixed
```

---

## Workflow

### Daily (15 min)
```
1. Scan news/sources
2. Pick best stories
3. AI writes summaries
4. Add personal commentary
5. Format and send
```

### Weekly (1 hour)
```
1. Plan theme
2. Research deeply
3. Write main piece
4. Add picks/tools
5. Format for platform
```

## Monetization

### 1. Sponsorships
```
Rate: $1 per subscriber
Example: 10K subs = $10K/episode
Types: Inline ads, dedicated sends
```

### 2. Premium Subscriptions
```
Price: $5-25/mo
Perk: Full articles, archives
Conversion: 3-8% typical
```

### 3. Affiliate Links
```
Products: Tools, courses, services
Placement: Within content
Commission: 10-30%
```

### 4. Own Product
```
Types: Courses, templates, consulting
Audience: Warm leads
Conversion: Higher than cold
```
