## Ramadan Content Guidelines

### Timing Adjustments
- Avoid peak posting 12:00-17:00 WIB (hot afternoon, people resting)
- Best times: 18:00-20:00 (after iftar) and 02:00-04:00 (sahur)
- Instagram stories: 20:00-22:00 WIB

### Content Themes
- "Produktif di bulan Ramadan dengan AI"
- "Optimalkan waktu sahur/buka dengan tools ini"
- "Ramadan adalah waktu terbaik untuk investasi diri"
- "AI membantu kamu tetap produktif sambil ibadah"

### Tone Adjustments
- More inspirational, less aggressive sales
- Include greetings: "Ramadan Kareem", "Marhaban Ya Ramadan"
- Reference spiritual growth alongside productivity

### Special Hashtags
`#ramadan2026` `#ramadankareem` `#ramadanproduktif` `#berkahramadan` `#sahurtime` `#bukapuasa`

## Harbolnas Content Guidelines

### 11.11 Strategy
- Start teasing 5 days before (Nov 6)
- Peak content on Nov 10-11
- Post at midnight Nov 10-11 for viral reach
- Flash sale mechanic: "hanya untuk X pembeli pertama"

### 12.12 Strategy
- Reuse 11.11 templates with new numbers
- Emphasize year-end clearance
- Bundle deal messaging: "Paket hemat akhir tahun"

### Key Messaging
- Urgency: "Limited time", "Countdown", "Jangan sampai kehabisan"
- Value: "Harga terbaik tahun ini", "Hemat [X]%"
- Social proof: Show previous buyers, review screenshots

## Media Asset Requirements

### Priority Matrix

| Asset Type | Platforms | Format | Priority |
|-----------|-----------|--------|----------|
| product_demo_video | TikTok, IG, YT | 9:16 MP4 | CRITICAL |
| tutorial_video | TikTok, IG | 9:16 MP4 | CRITICAL |
| tips_video | TikTok, YT | 9:16 MP4 | HIGH |
| promo_video | All | 9:16 MP4 | HIGH |
| carousel_images | Instagram | 1:1 JPG | HIGH |
| story_image | Instagram | 9:16 JPG | MEDIUM |
| product_image | All | 1:1 JPG | MEDIUM |
| testimonial_video | TikTok | 9:16 MP4 | MEDIUM |
| bts_video | TikTok, IG | 9:16 MP4 | LOW |

### Video Specs
- Resolution: 1080x1920 (9:16) for TikTok/IG/YT Shorts
- Resolution: 1920x1080 (16:9) for Facebook/YouTube
- Format: H.264 MP4
- Bitrate: 2-5 Mbps
- Frame rate: 30fps

### Image Specs
- Resolution: 1080x1080 (square) or 1080x1920 (portrait)
- Format: JPG or PNG
- Quality: 80-90%
- Max size: 8MB

## PostBridge API Quick Reference

```bash
# Get social accounts
curl -H "Authorization: Bearer REDACTED_ROTATED_CREDENTIAL" \
  https://api.post-bridge.com/v1/social-accounts

# Create scheduled post
curl -X POST -H "Authorization: Bearer REDACTED_ROTATED_CREDENTIAL" \
  -H "Content-Type: application/json" \
  -d '{
    "caption": "Your caption here #hashtag",
    "scheduled_at": "2026-03-14T07:00:00+07:00",
    "social_accounts": [48186]
  }' \
  https://api.post-bridge.com/v1/posts

# Get upload URL for media
curl -X POST -H "Authorization: Bearer REDACTED_ROTATED_CREDENTIAL" \
  -H "Content-Type: application/json" \
  -d '{"content_type": "video/mp4"}' \
  https://api.post-bridge.com/v1/media/create-upload-url
```
