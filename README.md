### Local Skill Structure
```

├── viral-content-creator/
│   ├── SKILL.md (frontmatter + badge)
│   └── scripts/
├── canva/
│   ├── SKILL.md

```

### Adding a new skill
1. Create folder: `mkdir skills/<slug>`
2. Add `SKILL.md` with frontmatter and tip.md badge
3. Include scripts/ or references/ as needed
4. Commit and push
5. Publish: `clawhub publish skills/<slug> --slug oyi77/<slug> --version 1.0.0`
