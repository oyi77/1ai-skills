Retry Decorator Test:
---------------------
Test Code:
```python
from trading.utils.error_handler import retry

call_count = 0
@retry(max_attempts=2, backoff=0.1)
def flaky_func():
    global call_count
    call_count += 1
    print(f'Called {call_count}')
    raise ValueError('fail')

try:
    flaky_func()
except:
    pass
print(f'Total calls: {call_count}')
```
