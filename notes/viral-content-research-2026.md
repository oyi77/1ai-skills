## 2. 🎬 HOOKS & FORMATS THAT CONVERT (Ranked)

### MASTER HOOK FORMULAS FOR INDONESIA

#### Tier 1: Highest Performing Hooks (Direct Revenue)

**1. "Penyesalan" Hook (Regret hook)**
```
"Kenapa gue ga tau ini dari dulu..."
"Gue menyesal butuh 2 tahun buat tau ini..."
"Kalau gue tau ini lebih cepat, gue udah..."
```
*Why it works:* Creates FOMO + triggers curiosity gap. Audience wants to avoid the same mistake.

**2. "Rahasia/Bocoran" Hook (Secret reveal)**
```
"Rahasia yang seller sukses ga mau kamu tau..."
"Bocoran: cara ini yang bikin mereka [HASIL]..."
"Yang ga diajarkan di sekolah tentang [TOPIK]..."
```
*Why it works:* Insider knowledge positioning. Indonesian culture values "orang dalam" (insider access).

**3. "Hasil Nyata" Hook (Real results)**
```
"Dari 0 ke [HASIL] dalam [WAKTU], ini caranya..."
"Bulan lalu gue hasilin [IDR X] dari HP doang..."
"Cara gue [HASIL] tanpa modal besar..."
```
*Why it works:* Proof-first selling. Indonesians are skeptical by default — show receipts early.

**4. "Masalah Kamu" Hook (Problem identification)**
```
"Kalau kamu [masalah spesifik], tonton ini dulu..."
"Stop [kebiasaan buruk]! Ini yang seharusnya kamu..."
"Kenapa [usaha kamu] ga berkembang (dan solusinya)"
```
*Why it works:* Direct identification creates "this is for ME" feeling = completion rate skyrockets.

**5. "Kontroversi Ringan" Hook (Mild controversy)**
```
"Gue ga setuju sama cara [umum] ini..."
"Semua orang salah tentang [topik]..."
"Hot take: [pendapat yang berbeda]..."
```
*Why it works:* Disagreement = comments flood = algo boost. Indonesian audience LOVES debat.

## 4. 🏆 VIRAL POST STRUCTURE ANALYSIS

### Example Structure A: EDUCATIONAL CAROUSEL (Instagram)
**Category:** "Tips bisnis digital" — consistently viral

```
SLIDE 1 (Hook):
- Visual: Bold text on high-contrast background
- Copy: "5 alasan produk digital kamu ga laku
  (dan cara fixnya 🔥)"
- CTA visual: arrow or swipe indicator

SLIDE 2-4 (Problem agitation):
- Each slide = 1 pain point
- Relatable language: "Pernah ngerasa..."
- Format: Icon + short headline + 1 sentence explanation

SLIDE 5-7 (Solution):  
- Mirror structure: "Cara fiksinya:"
- Simple, actionable, specific
- Each solution directly matches each problem

SLIDE 8 (Social proof):
- Screenshot/data/result
- "Setelah gue terapin ini, [spesifik hasil]"

SLIDE 9 (CTA):
- "Kalau kamu mau [hasil], link di bio 👆"
- Visual: product mockup or screenshot
- Sense of urgency if applicable

CAPTION:
- Line 1: Repeat hook (match slide 1)  
- Body: 3-5 bullet points of value
- CTA: "Save dulu biar ga lupa!" (saves = algo boost)
- Hashtags: 5-10 targeted (not maxed out)
```

---

### Example Structure B: SHORT VIDEO TIKTOK (15-30 detik)
**Category:** "Cara cepat [hasil]" — highest completion rate

```
0:00-0:03 (HOOK - text overlay):
"Cara gue hasilin 10 juta dari HP dalam seminggu 📱"
[Visual: dramatic start, direct to camera or screen recording]

0:03-0:10 (PROOF/CREDIBILITY):
Quick screenshot or visual evidence
"Ini bukan clickbait, ini faktanya..."

0:10-0:25 (CONTENT/VALUE):
3 steps maximum, very fast pace
Use text overlay + voiceover combo
Tempo: energetic, no dead air

0:25-0:30 (CTA):
"Link di bio kalau mau [hasil yang sama]"
OR "Follow untuk strategi lanjutannya"
[End card with profile picture]

CAPTION:
- Short, punchy (2-3 baris max)
- 1 question to trigger comments
- 3-5 hashtags
```

### Example Structure C: FACEBOOK STORYTELLING POST
**Category:** "Transformasi/journey" — high share rate

```
OPENING HOOK (baris 1-2, visible before "lihat selengkapnya"):

IMAGE:
- Before/after graphic
```

## 5. 💵 PRICE POINTS THAT CONVERT BEST

### Indonesian Consumer Psychology

**The "Magic Price Windows" (IDR):**

| Price Tier | Label | Conversion Rate | Best For |
|---|---|---|---|
| **FREE** | Lead magnet | Highest volume | List building, trust |
| **9K - 19K** | Impulse buy | Very high | Mini ebooks, presets |
| **29K - 49K** | 🎯 SWEET SPOT | **Highest conversion** | E-books, templates, trackers |
| **79K - 99K** | Considered purchase | Good | Mid-range courses, bundles |
| **149K - 199K** | Premium tier | Moderate | Full courses, toolkits |
| **299K - 499K** | High ticket | Low volume, high margin | Comprehensive programs |
| **999K+** | Premium/VIP | Niche | 1:1 coaching, masterclass |

### 🎯 KEY INSIGHT: The IDR 29K-49K Sweet Spot
- Below psychological "ragu" barrier (IDR 50K mental anchor)
- Feels like "sacrifice worthy" but not "risky"
- QRIS / GoPay / OVO payment = instant checkout, minimal friction
- Indonesian audience: **Price sensitivity is REAL** — social proof must do heavy lifting above IDR 99K

### Pricing Strategy Recommendations:

**1. FREE → PAID Funnel (Most Effective)**
```
FREE lead magnet (PDF/template) 
→ IDR 49K entry product (builds trust)
→ IDR 149K – 249K upgrade (premium)
→ IDR 499K+ (high ticket)
```

**2. Bundle Strategy (Increased AOV)**
```
Individual: IDR 49K each
Bundle 3 products: IDR 99K (saves 33%)
Bundle 5 products: IDR 149K (saves 39%)
```
*Bundle language: "Hemat IDR X kalo beli bundel sekarang"*

**3. Urgency Pricing**
```
"Harga early bird IDR 29K (normal IDR 79K) — 48 jam lagi!"
```
*Works extremely well in Indonesian market — FOMO is real*

## 9. ⚠️ COMMON MISTAKES TO AVOID

| Mistake | Why It Kills Conversions |
|---|---|
| Caption terlalu panjang tanpa value | Scroll away before CTA |
| Link di caption Instagram | Instagram doesn't make links clickable in captions |
| Terlalu hard sell dari awal | Indonesian audience needs trust-building first |
| Harga tanpa konteks | Always frame value first, then price |
| Posting tanpa konsistensi | Algorithm kills accounts with irregular posting |
| Mengabaikan komentar | Comments signal = algo boost + community trust |
| Generic hooks ("Hai guys!") | Zero stop power = zero completion |
| Same content cross-post tanpa adapt | Each platform has different native format expectations |
