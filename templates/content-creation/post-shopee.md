---

## 🖼️ Image Requirements

### Product Image (Main)
```
Resolution: 800x800 (square)
Background: Pure white (#FFFFFF)
Product occupies: 80-90% of frame
No text overlays (strict rule)
Multiple angles recommended (5-7 images)
```

### Shop Banner
```
Size: 1920x480 (mobile)
Format: JPG/PNG under 2MB
Design: Promotional, branded
Animation: Optional (5 sec max)
```

## ✍️ Product Title Template

```
[BRAND] [PRODUCT NAME] - [KEY BENEFIT] - [SPECS/FEATURES] - [PRICE RANGE]

Example:
"Nyamire Sepatu Dapur - Anti Slip - Size 39-43 - Bahan PVC Premium - IDR 59K"

✅ Do:
• Use keywords naturally
• Include benefits & features
• Add size/color options
• Keep it under 70 chars

❌ Don't:
• Capitalize every letter
• Use promotional words (TERLARIS, HOT)
• Overuse exclamation marks
```

## 📊 Flash Sale Strategy

```
Pre-event:
• Announcement 3 days before
• Countdown in feed
• Teaser image/video

During event:
• Update stock status hourly
• Real-time Q&A
• Multiple posts (don't spam)

Post-event:
• Thank you post
• Thank customers
• Teaser next event
```
