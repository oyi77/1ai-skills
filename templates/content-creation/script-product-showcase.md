## 📱 Script Structure (15 Second - High Engagement)

### Scene 1: Hook (0:00-0:02)
```
VISUAL: [Show product immediately - close-up]
TEXT: "[RESULT] IN [TIME]"
AUDIO: [Trending sound or upbeat music]

Example:
VISUAL: Camera zooms into the product
TEXT: "10 JUTA DI 7 HARI"
AUDIO: Trending upbeat track
```

### Scene 1: Problem/Pain Point (0:02-0:05)
```
VISUAL: [Show the problem this product solves]
TEXT: "[Problem in 6 words]"
AUDIO: [Music continues]

Example:
VISUAL: Old ineffective method shown
TEXT: "Cara lama yang capek"
AUDIO: Music continues
```

### Scene 1: Solution Reveal (0:05-0:10)
```
VISUAL: [Show THE PRODUCT in action]
TEXT: "[Product name + key benefit]"
AUDIO: [Voiceover OR music with text-to-speech]

Example:
VISUAL: Hands using the product perfectly
TEXT: "NYAMIRE - Cara Paling Efektif"
AUDIO: "Ini yang lebih efektif"
```

### Scene 1: Proof/Benefit Showcase (0:10-0:13)
```
VISUAL: [Quick cuts of results or happy customer]
TEXT: "[Result achieved]"
AUDIO: [Music + sound effect]

Example:
VISUAL: Before/after or happy customer
TEXT: "Hasil 3x Lebih Cepat"
AUDIO: Satisfying "ding" sound
```

### Scene 1: CTA (0:13-0:15)
```
VISUAL: [Show product + price OR pointing to link]
TEXT: "[Price] - SHOP NOW!"
AUDIO: [Voiceover OR music ends]

Example:
VISUAL: Product with price tag
TEXT: "IDR 59K - CEMUNGUD"
AUDIO: "CEMUNGUD!"
```

## 🎬 Full Production Notes

### Shooting Guidelines
```
• Lighting: Bright, natural light preferred
• Camera angle: Eye-level or slightly above
• Movement: Smooth, steady movements
• Background: Clean, branded (logo visible)
• Product: Clean, visible, 80% of frame
```

### Text Overlay Style
```
Font: Bold, sans-serif (Impact, Montserrat)
Size: Large, 80-100% of video width
Color: White text with black outline
Position: Center or upper third
Duration: 1-2 seconds per text change
```

---

## 📝 15 vs 30 Second Versions

### 15 Second (For TikTok)
```
Focus: Speed, hook, excitement
Script:
Hook → Problem → Product → Result → CTA
```

### 30 Second (For Instagram Reels)
```
Focus: Story, education, trust
Script:
Hook → Story → Problem → Product → Demo → Social Proof → CTA
```

## 🎵 Audio & Sound FX

### Voiceover Options
```

```

### Sound Effects
```
• "Ding" or "Ping" for reveals
```

## 📊 Performance Tracking

### Engagement Benchmarks
```
Views after 1 hour:
• 500+ = Good
• 2K+ = Viral potential

Engagement rate:
• 10%+ = Above average
• 25%+ = Viral territory

Save rate:
• 20%+ = High engagement
```

### Key Metrics to Watch
```
• Watch time (should be 80%+)
• Re-watch rate (repeat viewing)
• Comments (questions about product)
• Shares (virality indicator)
```
