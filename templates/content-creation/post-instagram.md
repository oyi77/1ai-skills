## 🖼️ Visual Specs

### Feed Post (1:1 or 4:5)
```
Resolution: 1080x1080 (square) or 1080x1350 (portrait)
File type: JPG/PNG
Max file: 30MB
Caption: 2,200 chars max
```

### Story (9:16)
```
Resolution: 1080x1920
Duration: 15 seconds (default)
Interactive stickers: poll, quiz, slider
CTA: "Swipe up" / "Link in bio"
```

### Reel (9:16)
```
Resolution: 1080x1920
Duration: 15-60 seconds (optimal)
Cover image: Best frame at 50% mark
Audio: Use trending sound
```
