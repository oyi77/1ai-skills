## 🎱 TikTok Ad Structure

### Full Video Script (15-30 Seconds)

#### Scene 1: Hook (0:00-0:03)
```
VISUAL: [Face to camera OR product close-up]
TEXT OVERLAY: "[HOOK - 4 words max, stops scroll]"
AUDIO: [Trending sound + VO or TTS]

Example:
VISUAL: My face with confident expression
TEXT: "STOP OVERPAYING FOR ADS"
AUDIO: Trending upbeat track + "Ada cara lebih murah..."
```

#### Scene 1: Problem (0:03-0:07)
```
VISUAL: [Show the problem/struggle]
TEXT: "[Problem in 6-8 words]"
AUDIO: [Continue audio]

Example:
VISUAL: Frustrated with expensive ads
TEXT: "Iklan boros tapi ga closing"
AUDIO: "Boros banget budget tapi..."
```

#### Scene 1: Solution (0:07-0:12)
```
VISUAL: [Show THE SOLUTION - product or service]
TEXT: "[Product name + 1 benefit]"
AUDIO: [VO or TTS with benefit]

Example:
VISUAL: TikTok Ads optimization tool
TEXT: "TOKOPEDIA ADS OPTIMIZER"
AUDIO: "Ini tool yang bikin iklanmu lebih efisien"
```

#### Scene 1: Proof/Demo (0:12-0:15)
```
VISUAL: [Quick demo or result]
TEXT: "[Key result]"
AUDIO: [Sound effect or continue audio]

Example:
VISUAL: Dashboard showing improved results
TEXT: "ROAS Naik 3x!"
AUDIO: Satisfying "ding" sound
```

#### Scene 1: CTA (0:15-0:15)
```
VISUAL: [Product + price OR pointing to link]
TEXT: "[CTA + urgency]"
AUDIO: [Strong CTA voice]

Example:
VISUAL: Product with limited offer badge
TEXT: "50% OFF HARI INI!"
AUDIO: "Checkout sekarang sebelum habis!"
```

## 🎨 Visual Guidelines

### Image/Video Specs
```
Resolution: 1080x1920 (9:16)
Duration: 15-30 seconds optimal
Format: MP4 (preferred) or MOV
File size: Under 500MB
Aspect ratio: 9:16 only (TikTok doesn't support others)
Caption: Recommended (sound-off viewing)
```

### Creative Best Practices
```
✅ Hook visible in first 0.5 seconds
✅ Text overlays large and readable
✅ High contrast (white text on dark bg)
✅ Fast-paced cuts (1-2 seconds per scene)
✅ Trending sound (in library)
✅ Natural, authentic feel (over-polished = suspicious)
❌ No vertical video
❌ No horizontal orientation
❌ No low-res images
❌ No stocky, generic footage
```

## 🎯 Spark Ads (Boost Organic Content)

### When to Use Spark Ads
- Already have viral/engaging content
- Want to scale organic reach
- Lower trust issues (user-generated content)
- Lower cost per engagement

### Setup Process
```
1. Create organic post
2. Monitor performance for 24-48h
3. If performance > 5% engagement rate
4. Turn into Spark Ad with custom targeting
```

### Targeting Parameters
```
Demographics: Age 18-35
Interests: [NICHE SPECIFIC INTERESTS]
Behaviors: Purchase behavior, device type
Custom Audiences: Website visitors, app users
```

## 🚨 Common TikTok Ads Mistakes

❌ **Mistake 1:** Not using trending sounds
✅ **Fix:** Check TikTok Creative Center for trending sounds weekly

❌ **Mistake 2:** Hook after 2+ seconds
✅ **Fix:** Make hook visible in first 0.5 seconds

❌ **Mistake 3:** Too much text on screen
✅ **Fix:** Less text, more visual storytelling

❌ **Mistake 4:** Not adding captions
✅ **Fix:** Always include captions (sound-off viewing is common)

❌ **Mistake 5:** Not reusing viral organic content
✅ **Fix:** Use Spark Ads for best-performing organic posts

❌ **Mistake 6:** No social proof
✅ **Fix:** Add customer testimonials or numbers ("10K+ happy customers")

## 🎬 Production Tips

### Audio & Sound
```
• Use TikTok's built-in trending sounds
• Voiceover: Speak naturally, energetic
• Text-to-speech: Use TikTok's TTS for variety
• Sound effects: Add "dings" and "whooshes" for impact
```

### Filming
```
• Use natural light or ring light
• Hold phone horizontal for horizontal (then crop to vertical)
• Keep camera steady (no shaky footage)
• Show face for trust (unless product-only necessary)
• Keep background clean (or branded)
```

### Editing
```
• Fast-paced: 1-2 seconds per scene
• Text overlays: Large, bold font
• Transitions: Minimal (let video speak)
• Color correction: Bright, vibrant
• Add brand logo subtly
```
