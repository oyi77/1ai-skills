## 🎯 Content Quality Checklist

### Content itself
- [ ] **Hook grabs attention** (First second stops the scroll)
- [ ] **Clear value communicated** (What do they get?)
- [ ] **Story flows logically** (Problem → Solution → Result)
- [ ] **Content length appropriate** (15-60s for video, concise for text)
- [ ] **Call-to-action is clear** (What should they do next?)
- [ ] **Brand voice consistent** (Matches your tone)
- [ ] **No grammatical/spelling errors**

### Visuals
- [ ] **Image/video quality is high** (Clear, sharp, professional)
- [ ] **Visual alignment matches platform** (9:16 for TikTok, etc.)
- [ ] **Text overlays readable** (Large font, high contrast)
- [ ] **Lighting is good** (Bright, faces visible)
- [ ] **Background is appropriate** (Clean or branded)
- [ ] **Audio is clear** (No background noise)
- [ ] **Captions included** (For sound-off viewing)
- [ ] **No copyrighted content** (Music, images, footage)

---

## 🎨 Platform Specific Checklists

### TikTok
- [ ] **Vertical format (9:16)**
- [ ] **Duration: 15-30 seconds optimal**
- [ ] **Trending sound used** (From TikTok library)
- [ ] **3-5 targeted hashtags** (Maximum 5 is best)
- [ ] **Caption under 150 chars** (Best practice)
- [ ] **Hook visible in first 0.5 seconds**
- [ ] **Text is bold and readable**
- [ ] **Face visible** (For most content types)
- [ ] **CTA: Follow/Save/Shop**

### Instagram Feed
- [ ] **Format: 1:1 or 4:5 ratio** (1080x1080 or 1080x1350)
- [ ] **Caption within 2,200 chars**
- [ ] **15-30 hashtags max** (Use mix of niche & popular)
- [ ] **Image high quality** (No pixelation)
- [ ] **Branding consistent** (Colors, fonts, logo)
- [ ] **Geo-tag added** (If relevant)
- [ ] **Account tags correct**
- [ ] **CTA clear** (Follow/Link in bio/DM)

### Instagram Stories & Reels
- [ ] **Format: 9:16 vertical**
- [ ] **Duration: 15-30s optimal (Reels)**
- [ ] **Trending audio** (Reels: from library)
- [ ] **Interactive stickers added** (Poll, quiz, slider)
- [ ] **Text overlays readable**
- [ ] **CTA: Swipe up / Link in bio**
- [ ] **Multiple posts** (3-6 stories per day recommended)

### Facebook
- [ ] **Post length: 125-150 chars ideal** (Mobile view)
- [ ] **3-5 hashtags max** (Facebook doesn't like spammy tags)
- [ ] **Image: 1200x628 for link posts**
- [ ] **Video: 3-5 minutes optimal**
- [ ] **Text overlay minimal** (Facebook scans for spam)
- [ ] **Post shared to groups** (If organic)
- [ ] **Page tag included** (If cross-promoting)
- [ ] **CTA: Click/Visit/Learn More**

### Facebook Ads
- [ ] **Pixel tracking setup**
- [ ] **Ad copy length: 125-150 chars** (Primary text)
- [ ] **Headline under 40 chars**
- [ ] **Description under 30 chars**
- [ ] **CTA button appropriate** (Shop Now/Learn More)
- [ ] **Campaign objective aligned**
- [ ] **Target audience verified**
- [ ] **Budget set correctly**

## 🔍 SEO & Hashtag Checklist

### Hashtag Strategy
- [ ] **3-5 core hashtags** (Long-tail, niche-specific)
- [ ] **Mix of broad + specific** (e.g., #marketing vs. #digitalmarketingtips)
- [ ] **No banned/restricted hashtags**
- [ ] **Relevant to content** (Not spamming popular tags)
- [ ] **Trackable for analysis** (Unique tags to track campaign)

### Keywords
- [ ] **Primary keyword in title/first sentence**
- [ ] **Keywords naturally placed** (Don't keyword stuff)
- [ ] **Description includes benefits**
- [ ] **Alt text or captions include keywords**

## 📊 Performance & Tracking

### Analytics Setup
- [ ] **UTM parameters added** (For link tracking)
- [ ] **Link correct** (Tested and working)
- [ ] **Pixel firing** (If using ads/retargeting)
- [ ] **Campaign tagged** (If part of larger campaign)

### Goal Alignment
- [ ] **Matches campaign objective**
- [ ] **Target audience addressed**
- [ ] **Value prop clear**
- [ ] **Expected outcome defined**

## 📝 Caption & Text Checklist

### Caption Quality
- [ ] **Hook in first line** (Stops the scroll)
- [ ] **Clear value stated** (What's in it for them?)
- [ ] **Short paragraphs** (1-2 sentences each)
- [ ] **Emojis used strategically** (2-3 max, not spammy)
- [ ] **No excessive capitalization**
- [ ] **No more than 1-2 exclamation marks**
- [ ] **Call-to-action included**
- [ ] **Grammar & spelling checked**

### Caption Structure
```
[HOOK - First line stops scroll]

[BODY - Expand on value/benefit]
• [Bullet points if info-heavy]

[BRIDGE - Connect to CTA]
[Bridge sentence]

[CTA - Direct and clear]

[P.S. - Optional closing thought]

#Hashtags #Here #Max5
```

## 🚨 Compliance & Safety

### Platform Policies
- [ ] **Content complies with platform TOS**
- [ ] **No misleading claims**
- [ ] **No prohibited content** (Violent, illegal, etc.)
- [ ] **Copyright free music/images** (Clear usage rights)
- [ ] **No spam techniques** (Fake engagement, bot comments)

### Brand Safety
- [ ] **No controversial or offensive content**
- [ ] **Appropriate language** (For your audience)
- [ ] **Respectful of competitors**
- [ ] **Legal compliance** (If industry-regulated)

---

## 💰 Commercial Content Check

### If Promotional:
- [ ] **Clear disclosure if sponsored** (#ad, #sponsored)
- [ ] **Pricing accurate** (If showing prices)
- [ ] **No false or misleading claims**
- [ ] **Availability current** (Product in stock)
- [ ] **Link leads to correct page** (Or correct contact info)

### If Discount/Promo:
- [ ] **End date specified** (If limited time)
- [ ] **Terms & conditions clear** (If restrictions apply)
- [ ] **Discount percentage accurate**
- [ ] **Not misrepresenting value**

## 📢 Engagement Strategy

### Pre-Publish Preparation
- [ ] **Identify engagement opportunities** (Questions, polls, quizzes)
- [ ] **Plan comment responses** (For first 30 minutes)
- [ ] **Tag relevant accounts** (If appropriate)
- [ ] **Share to communities** (If organic)

### Post-Publish Plan
- [ ] **Engage with comments immediately** (First 10-15 min)
- [ ] **Reply to all comments** (Within 2 hours)
- [ ] **Share to Stories** (If performing well)
- [ ] **Boost if high engagement** (Within first hour)

## 📝 Post-Publish Checklist

Immediately after publishing:

- [ ] **Monitor for the first 30 min** (Engagement rate)
- [ ] **Watch for comments** (Reply quickly)
- [ ] **Share to Stories** (If high engagement)
- [ ] **Add to campaign tracker** (Document performance)
- [ ] **A/B test variations planned** (If applicable)
- [ ] **Note what worked/didn't** (For future content)

---

## 🔄 Optimization Plan

After 24 hours:

- [ ] **Review performance metrics**
- [ ] **Identify what worked** (High engagement elements)
- [ ] **Plan next iteration** (What to test next)
- [ ] **Scale or pause accordingly**
