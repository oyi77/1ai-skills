## 📱 Full Script (15 Second Version)

### Scene 1: Hook (0:00-0:02)
```
VISUAL: [On camera face + text overlay]
TEXT: [EMOTIONAL HOOK - 4 words max]
AUDIO: [Music starts - build anticipation]

Example:
VISUAL: My face looking shocked
TEXT: "STOP SCROLLING RN"
AUDIO: "TikTok viral sound"
```

### Scene 1: Problem (0:02-0:05)
```
VISUAL: [Show problem/struggle]
TEXT: [Problem in 8 words max]
AUDIO: [Music continues, no voiceover]

Example:
VISUAL: Me struggling with the problem
TEXT: "Laptop lemot bikin kesel?"
AUDIO: [Beat drops here]
```

### Scene 1: Solution (0:05-0:12)
```
VISUAL: [Show THE SOLUTION]
TEXT: [Solution in 1 punchy sentence]
AUDIO: [Voiceover OR text-to-speech]

Example:
VISUAL: Screen capture of the fix
TEXT: "Ini cara fix dalam 2 menit"
AUDIO: "Ini yang sering kalian salah paham"
```

### Scene 1: Proof/Tutorial (0:12-0:15)
```
VISUAL: [Quick demo or result]
TEXT: [Result or key takeaway]
AUDIO: [Voiceover continues OR music]

Example:
VISUAL: Before/after or final result
TEXT: "Lihat bedanya!"
AUDIO: "Gini deh caranya"
```

### Scene 1: CTA (0:15-0:28)
```
VISUAL: [Point down OR profile button]
TEXT: [CTA - 5 words max]
AUDIO: [Voiceover OR music]

Example:
VISUAL: Finger pointing down
TEXT: "Follow buat lebih banyak!"
AUDIO: "Follow aku biar nggak ketinggalan!"
```

## 🎯 Hook Types (Choose 1)

### 1. Problem Hook
```
"STROP SROLLING IF YOUR [PROBLEM]"

Example:
"STOP SCROLLING IF YOUR LAPTOP IS SLOW"
```

### 2. Result Hook
```
"I GAINED 10K FOLLOWERS IN ONE WEEK"

Example:
"I MADE 10JUTA IN ONE WEEK"
```

### 3. Controversy Hook
```
"EVERYONE IS DOING THIS WRONG"

Example:
"YOU'VE BEEN DOING THIS WRONG YOUR ENTIRE LIFE"
```

### 4. Teaser Hook
```
"I'LL GIVE YOU [RESULT] IN [TIME]"

Example:
"I'LL TEACH YOU HOW TO [RESULT] IN [TIME]"
```

## ✅ Viral Checklist

- [ ] Hook in first 0.5 seconds
- [ ] Text overlays easy to read (large font)
- [ ] High contrast (white text on dark bg)
- [ ] Trending sound (in sound library)
- [ ] Face camera or relatable subject
- [ ] Clear problem → solution flow
- [ ] Strong CTA at end (follow/save/share)
- [ ] 3-5 targeted hashtags
- [ ] Optimal length (15-30s)
- [ ] Watch time > 15s (benchmark)

---

## 🎬 15 vs 30 Second Versions

### 15 Second (High Viral Potential)
- Fast-paced
- Single problem/solution
- 2-3 scenes max
- Hook-heavy

### 30 Second (Educational Value)
- More depth
- 2-3 solutions/options
- 4-5 scenes
- Higher save rate

## 📊 Performance Benchmarks

| Metric | Good | Viral |
|--------|------|-------|
| Views in 1h | 500+ | 5K+ |
| Engagement rate | 10%+ | 25%+ |
| Saved rate | 20%+ | 40%+ |
| Shares | 5%+ | 15%+ |

## 🔄 Iteration Formula

**Version 1: Original**
- Try the full script
- Track performance

**Version 2 (if under 1K views):**
- Change hook test different angle
- New text overlay style
- Different trending sound

**Version 3 (if still under 1K views):**
- Completely new angle
- Different topic within niche
- New CTA

---

*Key insight: TikTok algorithm tests each video on your followers first. If they engage, it goes to "For You" page. This is why hook & early engagement are CRITICAL.*
