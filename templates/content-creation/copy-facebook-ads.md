## 🎯 Ad Copy Structure

### Primary Text (Above Image/Video)
```
[HOOK - First 125 chars MUST stop the scroll]

[BODY - 3-5 lines max. Focus on benefits, not features]

[VALUE PROPOSITION - What can they achieve?]

[SOCIAL PROOF - If available]
• [Result 1]
• [Result 2]

[CTA - Direct and specific]
👉 [Action with clarity]

[URGENCY - If applicable]
⏰ [Limited time offer]
```

### Key Elements
```
✅ Hook: Emotional or curiosity-driven question
✅ Body: Clear benefits in simple language
✅ CTA: Specific action (Learn More/Shop Now)
✅ Length: 125-150 characters (mobile)
✅ Emoji: 2-3 max (use strategically)
❌ No spammy all-caps
❌ No excessive exclamation marks
```

## 🖼️ Creative Spec Guidelines

### Single Image
```
Ratio: 1:1 (square) or 4:5 (portrait)
Size: 1080x1080 or 1080x1350
Text: Max 20% of image area
Key info: Product shot + minimal text overlay
```

### Video
```
Ratio: 1:1 or 4:5 or 9:16
Duration: 15-30 seconds optimal
First 3 seconds: HOOK critical
Captions: Recommended (sound off viewing)
CTA: Clear in video + on-screen text
```

### Carousel
```
Cards: 2-10 images
Card order: Hook → Problem → Solution → Features → CTA
Each card: One clear message
Consistent branding throughout
```

## 🔍 Optimization Checklist

### Launch Day
- [ ] Check pixel tracking

### Day 2-3
- [ ] Check initial CTR (>1%)

- [ ] Deep-dive into audience targeting

- [ ] Test new creative variations

## 🚨 Common Mistakes to Avoid

❌ **Mistake 1:** Too much text in primary text
✅ **Fix:** Keep it under 150 characters

❌ **Mistake 2:** Generic, boring hooks
✅ **Fix:** Use emotional, curiosity-driven questions

❌ **Mistake 3:** No clear CTA
✅ **Fix:** Always tell them what to do next

❌ **Mistake 4:** Testing too many variables at once
✅ **Fix:** Test hook OR visual, not both simultaneously

❌ **Mistake 5:** Ignoring mobile viewers
✅ **Fix:** Test creative on mobile, add captions
