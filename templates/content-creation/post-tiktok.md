## 🎬 Video Structure (Larry Playbook Formula)

### 0:00 - 0:02 (Hook)
```
Visual: [Screen shaker effect / bold text / face camera]
Text Overlay: [EMOTIONAL HOOK - 4 WORDS MAX]
Audio: [Trending sound / music with beat drop]
```

### 0:02 - 0:08 (Problem)
```
Visual: [Show the pain point / struggle / problem face]
Text: [Describe the problem in 1 sentence]
Audio: [Continue music / voiceover "Kamu sering ngalamin ini?"]
```

### 0:08 - 0:15 (Solution)
```
Visual: [Show the solution / product / transformation]
Text: [The solution in 1 punchy sentence]
Audio: [Music shift / voiceover "Ini solusinya!"]
```

### 0:15 - 0:25 (Proof/Tutorial)
```
Visual: [Demo / result showcase / before-after]
Text: [Key point 1-2-3 or result]
Audio: [Voiceover explaining step-by-step or benefits]
```

### 0:25 - 0:28 (CTA)
```
Visual: [Pointing down / arrow / profile button flash]
Text: [CTA - "Follow for more!" / "Link di bio!"]
Audio: [Voiceover "Follow aku biar nggak ketinggalan!"]
```
