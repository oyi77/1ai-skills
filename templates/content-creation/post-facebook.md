## 🖼️ Visual Specs

### Photo Post
```
Resolution: 1200x630 (recommended)
Min size: 470x246
File type: JPG/PNG
Text in image: max 20% of area
```

### Video Post
```
Resolution: 1280x720 (16:9) or 1080x1920 (9:16)
Duration: 3-5 min (optimal)
File type: MP4/MOV
Autoplay: First 3 seconds critical
```

### Live Video
```
Resolution: 720p or 1080p
Duration: 10-60 min
Promotion: Post 1 hour before
Interaction: Respond to comments live
```

## 🔍 Engagement Checklist

### Before Posting
- [ ] Check spelling/grammar
- [ ] Test link (if included)
- [ ] Verify visual quality
- [ ] Include clear CTA
- [ ] Add relevant hashtags (3-5 max)
- [ ] Mention relevant pages/people

### After Posting (First Hour)
- [ ] Respond to all comments
- [ ] Like/boost high-quality comments
- [ ] Share to relevant groups (if organic)
- [ ] Pin best comment (if valuable)

## 🔄 Reposting Strategy

**Old content, new life:**
- Update headline/hook
- Refresh visuals (new crop/color)
- Update stats/data if outdated
- Add new CTA or offer
- Change posting time
