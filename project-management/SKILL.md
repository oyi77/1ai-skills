# project-management Skill

## What It Does

Task and project coordination - create tasks, track deadlines, manage sprints, coordinate team activities, and maintain project documentation.

## When to Use

- Create and manage projects
- Track tasks and deadlines
- Coordinate sprints
- Manage priorities
- Generate status reports

## Key Capabilities

- **Task Management**: Create, assign, track tasks
- **Sprint Planning**: Organize work into sprints
- **Deadline Tracking**: Monitor due dates and milestones
- **Priority Management**: Rank and reprioritize
- **Status Reporting**: Generate project updates

## Browser Workflows

### Create Task via Project Tool

1. Navigate: project management tool (Notion, Asana, Linear)
2. Click: "Add task" or "+" button
3. Fill: title, description, due date, priority
4. Assign: to appropriate category/sprint
5. Save: task is created

### Track Sprint Progress

1. Navigate: sprint board
2. View: current sprint tasks
3. Update: status (To Do → In Progress → Done)
4. Identify: blockers
5. Report: sprint status

## Usage Examples

### Create New Project
```
User: "Set up a new project for our Q2 product launch"
Skill: Creates project → adds milestones → creates initial tasks → sets timeline
```

### Plan Sprint
```
User: "Plan next sprint with these 10 features"
Skill: Prioritizes features → estimates effort → allocates tasks → sets sprint goals
```

### Generate Status Report
```
User: "What's the status of all active projects?"
Skill: Aggregates data → generates summary → highlights blockers → reports
```

## Priority Framework

| Priority | Definition | Response Time |
|----------|------------|---------------|
| P0 - Critical | Revenue impacted | Immediate |

| P2 - Medium | Normal work | 1 week |
| P3 - Low | Nice to have | Next sprint |
