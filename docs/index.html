Get Started

Install in seconds

Pick your editor and copy the command.

claude skill add 1ai-skills/1ai-skills

Requires Claude Code CLI. Skills auto-activate based on context.

git clone https://github.com/1ai-skills/1ai-skills .cursor/skills

Clone into your project's .cursor/skills directory. Cursor detects them automatically.

git clone https://github.com/1ai-skills/1ai-skills ~/.codex/skills

Clone into your Codex skills directory. Skills load on next session start.

git clone https://github.com/1ai-skills/1ai-skills ~/.gemini/skills

Clone into your Gemini CLI skills directory. Works with Gemini CLI 2.0+.

npm install -g 1ai-skills

Global npm install. Run 1ai-skills init in your project to set up.
