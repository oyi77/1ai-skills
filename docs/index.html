Frequently Asked Questions

What is 1ai-skills?

A library of 1338 production-grade SKILL.md files that force AI agents to follow real workflows instead of cutting corners. Every skill includes anti-rationalization tables, code examples, and verification checklists.

Which agents does it work with?

Any agent that reads markdown files: Claude Code, Cursor, OpenAI Codex, Gemini CLI, OpenCode, Windsurf, and more.

How was it built?

The entire library - 1338 skills, engineering rules, linting system, sync mechanism, and this website - was built by an AI agent. Every line of code, every verification check.

Is it free?

Yes. MIT licensed. Use in personal or commercial projects. No API keys required.

How do I contribute?

Fork the repo, add a SKILL.md file following the template, and submit a PR. The linting system validates your skill automatically.
