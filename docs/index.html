Get Started

Quick Install

Install 1ai-skills in under a minute. Works with any AI agent harness that supports skill loading.

One-Command Install

Recommended

Install via npx -- works with Claude Code, Cursor, and other agent harnesses.

npx skills add oyi77/1ai-skills

Clone & Install

Manual

Clone the repository and install locally for development or customization.

git clone https://github.com/oyi77/1ai-skills.git
cd 1ai-skills && npm install

Browse Catalog

Explore

Query the machine-readable skill catalog to find exactly what you need.

cat SKILLS.json | jq '.[] | .name'
