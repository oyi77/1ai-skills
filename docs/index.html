Self-Evolving System

12 meta-skills that continuously improve the entire ecosystem. Skills learn from usage, adapt to new patterns, and optimize themselves over time.

🧠

Auto-Evolve Engine

Automatically identifies skill improvement opportunities based on usage patterns, error rates, and user feedback. Proposes and validates upgrades.

📊

Performance Monitor

Tracks skill execution metrics, success rates, and latency. Flags underperforming skills for review and optimization.

🧚

Auto-Learner

Extracts reusable patterns from successful skill executions and synthesizes new skill variants to fill coverage gaps.

📦

Skill Datastore

Central registry that versions, indexes, and distributes skills across agent environments. Supports hot-reload and rollback.

🔍

Gap Analysis

Scans the ecosystem for missing capabilities by comparing skill coverage against common agent task types and industry domains.

⚡

Conflict Resolver

Detects and resolves skill conflicts when multiple skills match the same intent. Applies priority rules and context-aware selection.

Quick Start

Install 1ai-skills in under a minute. Works with any AI agent harness that supports skill loading.

Step 1

Install via npx

npx skills add oyi77/1ai-skills
Step 2

Or clone directly

git clone https://github.com/oyi77/1ai-skills.git
cd 1ai-skills && npm install
Step 3

Browse the catalog

cat SKILLS.json | jq '.[] | .name'
