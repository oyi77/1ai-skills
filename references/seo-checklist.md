# SEO Checklist

Shared reference for all SEO-related skills. Load this when doing SEO work.

## Technical SEO
- [ ] Robots.txt exists and allows crawling
- [ ] XML sitemap exists and is valid
- [ ] No broken internal links (3xx/4xx/5xx)
- [ ] Canonical tags implemented correctly
- [ ] Hreflang tags for multi-language sites
- [ ] Open Graph / Twitter Card meta tags present
- [ ] Structured data (JSON-LD) valid and present
- [ ] No orphan pages (all pages reachable from sitemap or nav)

## On-Page SEO
- [ ] Title tags: unique, 50-60 chars, include target keyword
- [ ] Meta descriptions: 150-160 chars, compelling, unique per page
- [ ] H1: single per page, includes target keyword
- [ ] H2/H3: logical hierarchy, keywords included naturally
- [ ] URL structure: short, descriptive, hyphen-separated
- [ ] Image alt text: descriptive, keywords where relevant
- [ ] Internal linking: contextual links to related content
- [ ] External links: to authoritative sources, nofollow where appropriate

## Content Quality (E-E-A-T)
- [ ] Content demonstrates Experience, Expertise, Authoritativeness, Trust
- [ ] Word count appropriate for topic (1500+ for competitive keywords)
- [ ] Content is original, not duplicate or near-duplicate
- [ ] Freshness: content updated within last 12 months
- [ ] Sources cited with links to authoritative references
- [ ] Author bio with credentials displayed
- [ ] About page and Contact page present and complete

## Core Web Vitals
- [ ] LCP (Largest Contentful Paint) < 2.5s
- [ ] INP (Interaction to Next Paint) < 200ms
- [ ] CLS (Cumulative Layout Shift) < 0.1
- [ ] First Contentful Paint (FCP) < 1.8s
- [ ] Time to First Byte (TTFB) < 800ms

## AI Search Optimization (GEO)
- [ ] Content is passage-level citability (concise answers to specific questions)
- [ ] llms.txt file present at root (https://llmstxt.org/)
- [ ] GPTBot, ClaudeBot, PerplexityBot allowed in robots.txt
- [ ] Content structured for featured snippets (lists, tables, definitions)
- [ ] Brand mentions and citations tracked

## Local SEO (if applicable)
- [ ] Google Business Profile claimed and verified
- [ ] NAP (Name, Address, Phone) consistent across directories
- [ ] Local citations: 50+ high-quality directories
- [ ] Reviews: 4+ stars, responded to within 48 hours
- [ ] Location pages for each service area

## Analytics & Tracking
- [ ] Google Search Console verified
- [ ] Google Analytics 4 installed
- [ ] Conversion tracking configured
- [ ] No PII in URL parameters
- [ ] Event tracking for key interactions

## Common Red Flags
- Keyword stuffing (keyword density > 2%)
- Thin content (< 300 words on important pages)
- Hidden text or links
- Paid links without nofollow/ sponsored tags
- Auto-generated content without human review
- Negative SEO: sudden spike in toxic backlinks
