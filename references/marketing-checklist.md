# Marketing Checklist

Shared reference for marketing-related skills. Load this when doing marketing work.

## Campaign Planning
- [ ] Campaign objective: specific, measurable, time-bound
- [ ] Target audience: personas defined with demographics/psychographics
- [ ] Value proposition: clear, differentiated, compelling
- [ ] Budget allocated with ROI targets
- [ ] Timeline with milestones and deliverables
- [ ] Success metrics defined (KPIs)

## Content Marketing
- [ ] Content calendar: 4+ weeks planned ahead
- [ ] Content mix: educational (40%), promotional (20%), engaging (40%)
- [ ] SEO optimized: keywords, meta tags, internal links
- [ ] Content formats: blog, video, infographic, carousel varied
- [ ] Call-to-action on every piece
- [ ] Content repurposed across 3+ channels

## Social Media
- [ ] Posting frequency: daily (Twitter), 3-5x/week (others)
- [ ] Engagement rate tracked (target: 2%+ for organic)
- [ ] Response time: < 4 hours for comments/DMs
- [ ] Hashtag strategy: 3-5 relevant hashtags per post
- [ ] Visual consistency: brand colors, fonts, style
- [ ] Stories/Reels: daily for Instagram, 3x/week for LinkedIn

## Email Marketing
- [ ] List segmentation: by behavior, demographics, lifecycle stage
- [ ] Subject lines: < 50 chars, curiosity + benefit, no spam words
- [ ] Preview text optimized (preheader)
- [ ] Mobile optimization: 320px width tested
- [ ] CTA: single, clear, above the fold
- [ ] A/B tests: subject lines, send times, CTA buttons
- [ ] Unsubscribe link visible and functional
- [ ] Deliverability: sender score > 80, spam score < 2

## Paid Advertising
- [ ] Ad copy: benefit-focused, includes keyword, CTA present
- [ ] Landing page: matches ad promise, fast load, mobile-optimized
- [ ] Conversion tracking: pixel installed, events firing correctly
- [ ] Audience targeting: lookalike audiences from customer data
- [ ] Budget pacing: daily spend within 10% of target
- [ ] ROAS tracking: target > 3x for prospecting, > 5x for retargeting
- [ ] Negative keywords: 50+ added monthly

## Analytics & Reporting
- [ ] Dashboards: traffic, conversions, revenue, ROAS visible
- [ ] Attribution model: multi-touch (not last-click only)
- [ ] Anomaly detection: alerts for 20%+ drops/spikes
- [ ] Competitor benchmarking: monthly
- [ ] Report cadence: weekly for campaigns, monthly for strategy

## Growth Engineering
- [ ] Viral coefficient tracked (K-factor = invites per user × conversion rate)
- [ ] Referral program: double-sided incentive, seamless sharing
- [ ] A/B tests: minimum 1000 visitors per variant before decision
- [ ] Onboarding funnel: < 3 steps to value, < 60 seconds
- [ ] Retention cohorts: Day 1, Day 7, Day 30 tracked
- [ ] Churn analysis: reasons categorized, win-back campaigns active

## Common Red Flags
- High CPC with low conversion (fix landing page or kill keyword)
- Email open rates < 15% (refresh subject line strategy)
- Social engagement < 1% (content not resonating, pivot)
- Content without CTAs (traffic with no conversions)
- No attribution tracking (flying blind on ROAS)
- Viral coefficient < 1.0 (product not worth sharing)
