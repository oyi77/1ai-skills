## Blog Posts

### Research Phase
Before writing, gather:
- Target keyword + 3-5 semantic variations
- Top 5 SERP results — analyze structure, word count, angle
- "People Also Ask" questions to address
- Unique angle or data the competition doesn't have

### Structure Template

```markdown
# {H1: Keyword-rich, compelling, under 60 chars}

{Hook paragraph: Open with a stat, question, or bold statement. 2-3 sentences max.}

## {H2: First major section}
{Content with short paragraphs (2-4 sentences). Use subheadings every 200-300 words.}

### {H3: Sub-section if needed}

## {H2: Second major section}

## Key Takeaways
{Bulleted summary of main points — aids scannability and featured snippets}

## {CTA Section}
{Clear next step aligned with funnel stage}
```

### Word Count Guidelines
| Content Type | Target Length | Reasoning |
|-------------|--------------|-----------|
| Pillar / Guide | 2,500-4,000 | Comprehensive coverage, link magnet |
| How-to / Tutorial | 1,500-2,500 | Step-by-step depth |
| List post | 1,200-2,000 | Scannable, shareable |
| Opinion / Thought | 800-1,500 | Voice-driven, engagement |
| News / Update | 500-800 | Timely, to the point |

### SEO Integration
- Primary keyword in: H1, first 100 words, one H2, meta description, URL slug
- Keyword density: 1-2% (never force it)
- Semantic keywords: sprinkle naturally throughout
- Internal links: 3-5 to related content
- External links: 2-3 to authoritative sources
- Image alt text: descriptive, keyword-relevant where natural

## Social Media

### Platform-Specific Formats

#### LinkedIn
- **Post length:** 150-300 words sweet spot (can go to 3,000 char limit)
- **Hook:** First 2 lines must compel "See more" click
- **Structure:** Hook → Story/Insight → Takeaway → CTA → Hashtags
- **Hashtags:** 3-5 relevant ones at end
- **Tone:** Professional but human, data-backed insights work well
- **Carousel:** 8-12 slides, one idea per slide, bold headlines

#### Instagram
- **Caption:** 125-150 chars visible before fold; front-load value
- **Hashtags:** 5-10 (mix of broad + niche)
- **Stories:** 3-7 slides, mix of text/visual, end with engagement sticker
- **Reels:** 30-90 seconds, hook in first 3 seconds, captions on

#### Twitter/X
- **Tweet:** Under 280 chars; threads for depth
- **Thread structure:** Tweet 1 = hook + promise; 2-8 = value; last = CTA + recap
- **Images increase engagement ~150%**

#### TikTok
- **Hook in first 1-3 seconds** — text overlay + visual disruption
- **Length:** 30-60 seconds (60-90 for educational)
- **Structure:** Hook → Context → Value → CTA
- **Caption:** Short, keyword-rich, 3-5 hashtags

#### Facebook
- **Post length:** 40-80 words for organic; longer for groups
- **Engagement bait to AVOID:** "Tag a friend," "Share if you agree"
- **What works:** Questions, stories, behind-the-scenes, polls

### Content Pillars Strategy
Help the user define 3-5 content pillars:
1. **Educational** — How-tos, tips, tutorials (40% of content)
2. **Inspirational** — Success stories, transformation (20%)
3. **Entertaining** — Memes, trends, relatable content (15%)
4. **Promotional** — Product features, offers, launches (15%)
5. **Community** — UGC, polls, Q&A, behind-the-scenes (10%)

## Landing Pages

### Hero Section Formula
```
{Pre-headline: Category or social proof — "Trusted by 10,000+ teams"}
{Headline: Clear value proposition — what they GET}
{Sub-headline: How it works in one sentence + who it's for}
{CTA Button: Action verb + value — "Start Free Trial" not "Submit"}
{Supporting visual: Product screenshot, demo video, or hero image}
```

### Page Structure (Top to Bottom)
1. **Hero** — Value prop + primary CTA
2. **Social proof bar** — Logos, review scores, user count
3. **Problem section** — Agitate the pain they feel
4. **Solution section** — How you solve it (feature highlights)
5. **How it works** — 3-step process (simplicity signal)
6. **Benefits** — Outcome-focused (not feature-focused)
7. **Testimonials** — Specific results, named customers
8. **Pricing** (if applicable) — Anchor with popular plan
9. **FAQ** — Objection handling disguised as helpfulness
10. **Final CTA** — Repeat hero CTA with urgency/scarcity element

### CTA Best Practices
- Use first-person: "Start my free trial" > "Start your free trial"
- Add value/benefit: "Get instant access" > "Sign up"
- Reduce friction: "No credit card required" near the button
- Urgency when honest: "Limited spots" only if true
- One primary CTA per viewport; secondary CTAs are fine but visually subordinate

## Ad Copy

### Meta (Facebook/Instagram) Ad Copy Templates

**Primary Text Formats:**

1. **Problem-Solution (PAS)**
```
Tired of {pain point}?

{Agitate: What happens if they don't solve it}

{Product} helps you {benefit} in {timeframe}.

{Social proof: "Join 5,000+ {target audience} who..."}

→ {CTA}
```

2. **Story-based**
```
{Name} was struggling with {problem}.
Then they discovered {product/method}.
Within {timeframe}, they {specific result}.

Here's what changed: {key insight}

Want the same results? {CTA}
```

3. **Direct-response**
```
🎯 {Bold claim or stat}

{1-2 sentences of context}

✅ {Benefit 1}
✅ {Benefit 2}
✅ {Benefit 3}

{Urgency element}
{CTA}
```

### Google Ads Copy Rules
- **Headline 1:** Include primary keyword (30 char limit)
- **Headline 2:** Unique value prop or benefit (30 char)
- **Headline 3:** CTA or brand (30 char)
- **Description 1:** Expand on value prop (90 char)
- **Description 2:** Social proof + CTA (90 char)
- Always write 3+ variants for Responsive Search Ads

### Always generate 3-5 copy variants for A/B testing.

## Video Scripts

### Explainer Video Template (60-90 seconds)
```
[0-5s]  HOOK: Start with the problem or a bold question
[5-15s] PROBLEM: "You know that feeling when..."
[15-30s] SOLUTION: "That's why we built..."
[30-50s] HOW IT WORKS: 3 simple steps
[50-65s] PROOF: Results, testimonials, numbers
[65-80s] CTA: "Try it free / Sign up today"
[80-90s] BRAND: Logo, tagline
```

### UGC/Testimonial Script Template
```
[Hook] "I almost didn't try {product}, but..."
[Problem] "I was dealing with {specific pain}..."
[Discovery] "Then I found {product}..."
[Result] "Now I {specific positive outcome}..."
[CTA] "If you're struggling with {problem}, just try it."
```

## Content Repurposing

### One-to-Many Matrix

| Source | Repurpose Into |
|--------|---------------|
| Blog post | → 5-8 social posts, 1 email newsletter, 1 infographic, 1 video script |
| Podcast/Webinar | → Blog recap, quote graphics, audiogram clips, Twitter thread |
| Case study | → Social testimonial posts, email proof point, ad creative |
| Data/Report | → Infographic, social stats carousel, press pitch, blog analysis |
| Customer review | → Social proof post, ad copy, website testimonial, email element |

### Repurposing Workflow
1. Extract key insights, quotes, data points from source
2. Adapt message depth per platform (LinkedIn = detailed, Twitter = punchy)
3. Rewrite — don't just truncate. Each platform has its own voice.
4. Add platform-native elements (hashtags, mentions, polls, carousels)
5. Stagger publishing across 1-2 weeks for maximum reach

## Quality Checklist

Before delivering any content, verify:

- [ ] **Audience match** — Written for the right persona and funnel stage
- [ ] **Brand voice** — Consistent with established guidelines
- [ ] **CTA present** — Clear, specific, and aligned with objective
- [ ] **SEO optimized** — Keywords natural, meta data complete (for web content)
- [ ] **Readability** — Appropriate for audience (aim Flesch-Kincaid grade 6-8 for general)
- [ ] **No jargon** — Unless audience expects it (B2B technical is fine)
- [ ] **Proofread** — Grammar, spelling, punctuation
- [ ] **Platform specs** — Character limits, image sizes, format requirements met
- [ ] **Unique angle** — Not generic AI slop; has a distinct point of view
- [ ] **Facts verified** — Statistics cited, claims supportable
