# Marketing Decision Engine

Solo founders don't fail from lack of knowledge — they fail from decision fatigue.
This reference provides decision frameworks so you spend less time deciding
and more time executing.

---

## Channel Selection: The Bullseye Framework

### Step 1: Brainstorm (5 minutes)
List all possible marketing channels:
```
□ SEO/Content    □ Paid Search    □ Paid Social    □ Email
□ Cold outreach  □ PR/Media       □ Partnerships   □ Affiliates
□ Community      □ Influencer     □ Social organic  □ Video/YouTube
□ Podcast        □ Events         □ Referral       □ Marketplace
□ Product Hunt   □ Reddit         □ WhatsApp       □ TikTok
```

### Step 2: Rank (10 minutes)
Score each channel on 3 criteria (1-5):
- **Reach:** Can this reach my ICP at volume?
- **Cost:** Can I afford to test this channel?
- **Speed:** How fast will I see results?

### Step 3: Test (2-4 weeks per channel)
Pick top 3 channels. Run minimum viable tests:
- Spend $200-500 or 10 hours per channel
- Measure: leads generated, cost per lead, quality
- After 4 weeks: rank by CPA and volume potential

### Step 4: Focus (Ongoing)
- **Inner ring:** 1 channel — 70% of budget/time (your proven winner)
- **Middle ring:** 1-2 channels — 20% (testing for next winner)
- **Outer ring:** Everything else — 10% (experiments only)

## Experiment Prioritization: ICE Scoring

For every marketing idea, score before executing:

| Dimension | Question | Score 1-10 |
|-----------|----------|-----------|
| **Impact** | If this works, how much will it move the needle? | |
| **Confidence** | How sure am I this will work? (data > gut) | |
| **Ease** | How quickly/cheaply can I test this? | |
| **ICE Score** | (I + C + E) ÷ 3 | |

**Priority rules:**
- ICE > 7: Do this week
- ICE 5-7: Add to next sprint
- ICE 3-5: Backlog — revisit later
- ICE < 3: Kill it

## Quarterly OKRs (Marketing)

### OKR Framework for Solo Founders

**Set 2 Objectives with 3 Key Results each. Track weekly.**

```markdown
## Q{X} Marketing OKRs

### Objective 1: {Growth-focused}
Example: "Build a predictable lead generation engine"
  KR1: Increase website traffic from {X} to {Y}/month
  KR2: Achieve {Z} qualified leads per month
  KR3: Reduce CAC from ${A} to ${B}

### Objective 2: {Efficiency-focused}
Example: "Maximize revenue from existing customers"
  KR1: Improve trial-to-paid conversion from {X}% to {Y}%
  KR2: Achieve Net Revenue Retention of {Z}%
  KR3: Launch referral program generating {N} new customers/month
```

### OKR Rules
- 2 objectives max (you're one person)
- Each KR must be measurable with a number
- Set targets that are ambitious but not impossible (70% achievement = healthy)
- Review weekly, adjust quarterly
- If you hit 100% of all OKRs, you set them too easy

## The "Should I Do This?" Decision Tree

```
Someone suggests a marketing tactic / you see a cool idea
                    ↓
        Does my ICP actually use this channel?
           NO → SKIP (no matter how trendy)
           YES ↓
        Can I test this in <2 weeks with <$500?
           NO → BACKLOG (revisit when you have resources)
           YES ↓
        Is my current top channel maxed out?
           NO → KEEP OPTIMIZING current channel first
           YES ↓
        Does this have a clear success metric?
           NO → DEFINE one, then test
           YES ↓
        ICE score > 5?
           NO → SKIP for now
           YES → TEST IT
```

## Time Allocation by Stage

### Where Your Marketing Hours Should Go

**Stage 1 ($0-1K MRR): 8 hours/week**
```
40% — Customer conversations (interviews, sales calls)
30% — Content creation (blog, social)
20% — Outreach (cold emails, community engagement)
10% — Setup (tools, tracking, profiles)
```

**Stage 2 ($1K-10K MRR): 10 hours/week**
```
30% — Content creation (your #1 channel)
25% — Paid acquisition testing + optimization
20% — Email marketing + automation
15% — Customer conversations + research
10% — Analytics + reporting
```

**Stage 3 ($10K-50K MRR): 12 hours/week**
```
25% — Scaling winning channels
25% — Content + SEO
20% — Campaign management
15% — Retention marketing
15% — Analytics + experimentation
```
