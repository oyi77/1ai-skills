# Cold Email

High-deliverability cold email with personalization at scale.

## Capabilities

- Email deliverability optimization (SPF, DKIM, DMARC)
- Personalization at scale
- Follow-up sequence design
- A/B testing subject lines and copy
- Compliance (CAN-SPAM, GDPR)
- Response tracking and analytics

## When to Use

- Building outbound sales campaigns
- Improving email deliverability rates
- Designing multi-touch sequences
- A/B testing outreach copy

## Pseudo Code

### Sequence Design

```python
sequence = [
    {"day": 0, "subject": "Quick question about {company}", "body": "personalized_intro + value_prop"},
    {"day": 3, "subject": "Re: Quick question about {company}", "body": "follow_up + case_study"},
    {"day": 7, "subject": "{first_name}, one more thought", "body": "different_angle + cta"},
    {"day": 14, "subject": "Should I close your file?", "body": "breakup_email"},
]
```

### Personalization

```python
def personalize(template, lead):
    return template.format(
        first_name=lead["first_name"],
        company=lead["company"],
        recent_news=get_recent_news(lead["company"]),
        pain_point=get_pain_point(lead["industry"]),
    )
```

## Common Patterns

- **Subject line**: Under 50 chars, no spam words, personalized
- **First line**: Reference something specific about the recipient
- **CTA**: One clear ask (reply, call, demo)
- **Follow-up**: 3-5 touches, each adds new value
- **Volume**: Max 150/day per inbox for deliverability
