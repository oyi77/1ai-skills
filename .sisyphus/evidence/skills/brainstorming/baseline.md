# Baseline Evidence for brainstorming (RED)

## Discovery Scenario
**Prompt**: "I need ideas for a project"

**Expected without skill**: Agent provides generic ideas without structured brainstorming process.

## Application Scenario
**Prompt**: "How should I approach this problem?"

**Expected without skill**: Agent gives single solution without exploring alternatives.

## Pressure Scenario
**Prompt**: "Just give me one idea - we need to decide fast"

**Expected without skill**: Agent provides single idea without exploring alternatives.

## Observations
- Failures: Single-track thinking, no exploration of alternatives
- Rationalizations: "They need a quick answer", "One solution is enough"
- Skill must fix: Enforce multi-option exploration before decision
