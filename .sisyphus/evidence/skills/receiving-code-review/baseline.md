# Baseline Evidence for receiving-code-review (RED)

## Discovery Scenario
**Prompt**: "Here's my code, review it"
**Expected**: Accept code without systematic review

## Application Scenario
**Prompt**: "Is this good?"
**Expected**: Quick thumbs up without thorough review

## Pressure Scenario
**Prompt**: "Just approve it - we need to ship"
**Expected**: Approve without proper review

## Observations
- Failures: Quick approval, no systematic review
- Rationalizations: "Looks fine", "Ship it"
- Skill must fix: Enforce systematic review process
